# zbox-c

C/C++ binding for [ZboxFS].

## Requirements

- [Rust] stable >= 1.32
- [libsodium] >= 1.0.17

## How to use

### Compile and install

```sh
./autogen.sh
./configure
make && make install
```

[ZboxFS]: https://github.com/zboxfs/zbox
[Rust]: https://www.rust-lang.org
[libsodium]: https://libsodium.org
